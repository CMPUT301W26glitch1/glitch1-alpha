## About The Project

Event Lottery System App is a mobile application that allows organizers to create and manage events, entrants to browse events, join waiting lists, receive notifications, and enroll in events, and administrators to manage events, users, and system content.

This project was developed as part of the University of Alberta CMPUT 301 course by **Glitch1**.

## Features

### Entrant Features
- Browse public events
- Search and filter events
- Join and leave event waiting lists
- Receive event notifications
- Accept or decline invitations
- View event details and lottery-related updates

### Organizer Features
- Create and manage events
- View participants by status
- Invite entrants to private event waiting lists
- Send notifications to selected participant groups
- Export enrolled participants as CSV
- Generate and view QR codes for events
- Manage event posters and comments

### Admin Features
- Manage events
- Manage organizers
- Manage entrant/admin profiles
- View administrative event details and logs

### Accessibility / WOW Factor
- Accessibility mode with larger text and buttons across key screens
